## Technologies

Initial implementation: HTML, SCSS, Native JS (then the project was transferred to react)

Modern version: 

-  React (Functional Components + Hooks)
-  Vite 5.2.0
-  Redux Toolkit
-  React Router v6 
-  Axios
-  Formik + Yup (forms)
-  Framer-motion (pages animation)
-  Swiper.js (sliders)
-  Leaflet (interactive maps)
-  Json-server (backend imitation)
